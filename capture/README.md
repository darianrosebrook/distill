# Capture

Trace capture and normalization utilities.

## Overview

This directory contains utilities for:
- Tool trace capture
- Trace normalization
- Proxy server for trace interception
- Trace validation

## Files

- `proxy_server.py` - Proxy server for capturing tool traces
- `normalize_trace.py` - Trace normalization utilities
- `validators.py` - Trace validation
